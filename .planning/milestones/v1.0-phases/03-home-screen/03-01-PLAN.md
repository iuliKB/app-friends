Create `src/hooks/useHomeScreen.ts` — data fetching hook:
- Import `supabase` from `@/lib/supabase`, `useAuthStore` from `@/stores/useAuthStore`, `useHomeStore` from `@/stores/useHomeStore`, `FriendWithStatus` from `@/hooks/useFriends`, `StatusValue` and `EmojiTag` from `@/types/app`
- `fetchAllFriends` async function:
  1. Guard: if `!session?.user` return early
  2. Call `supabase.rpc('get_friends')` to get all friends
  3. Extract `friendIds` array from result rows
  4. If `friendIds.length > 0`, query `supabase.from('statuses').select('user_id, status, context_tag, updated_at').in('user_id', friendIds)`
  5. Build `statusMap` as `Map<string, { status, context_tag, updated_at }>` from statuses result
  6. Map friend rows to `FriendWithStatus[]` array, defaulting status to `'maybe'` and context_tag to `null` if not found in statusMap
  7. Call `setFriends(allFriends)` to update Zustand cache
  8. Return the `allFriends` array (needed for subscription setup in Plan 02)
- `useEffect` on mount (`[session?.user?.id]` deps) calls `fetchAllFriends()`
- Derived arrays computed from `friends` store value:
  - `freeFriends`: filter where `status === 'free'`, sort by `updated_at` descending (use statusMap stored alongside or re-derive from a secondary map). Since `FriendWithStatus` doesn't have `updated_at`, extend the store to also cache a `statusUpdatedAt: Record<string, string>` map. Add `statusUpdatedAt` field to HomeState and `setStatusUpdatedAt` action. In `fetchAllFriends`, build this map from the statuses query `updated_at` field. Sort freeFriends using this map.
  - `otherFriends`: filter where `status !== 'free'`, sort by status order (Maybe=1, Busy=2) then `display_name.localeCompare()`. Use sort order map: `{ free: 0, maybe: 1, busy: 2 }`.
- `loading` state: `useState(false)`, set true at start of fetchAllFriends, false after
- `error` state: `useState<string | null>(null)`, set from catch block or RPC error
- `refreshing` state: `useState(false)` for pull-to-refresh
- `handleRefresh` function: sets `refreshing(true)`, awaits `fetchAllFriends()`, sets `refreshing(false)`
- Return object: `{ friends, freeFriends, otherFriends, loading, error, refreshing, handleRefresh, fetchAllFriends }`
- Named export: `useHomeScreen`

Note: Do NOT add Realtime subscription in this plan — that is Plan 02.
