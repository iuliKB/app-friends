2. Create `src/app/profile/edit.tsx`:
- Default export `EditProfileScreen`
- State: `displayName` (string, init from fetched profile), `avatarUrl` (string|null, init from fetched profile), `originalDisplayName` (to detect changes), `saving` (boolean), `avatarLoading` (boolean), `loading` (boolean for initial fetch)
- On mount useEffect: fetch current profile from `supabase.from('profiles').select('display_name, avatar_url').eq('id', session.user.id).single()` and populate state
- Show `<LoadingIndicator />` while loading initial profile data
- Avatar section (centred): `<AvatarCircle size={80} imageUri={avatarUrl} displayName={displayName || 'U'} onPress={handleChangeAvatar} />`
  - When `avatarLoading`: show `<ActivityIndicator>` overlay on avatar area and change label to "Uploading..."
  - Below avatar: `<TouchableOpacity onPress={handleChangeAvatar}><Text style={{ fontSize: 14, color: COLORS.accent }}>Change Photo</Text></TouchableOpacity>` (or "Uploading..." when avatarLoading)
- `handleChangeAvatar`: `Alert.alert('Change Photo', undefined, [...])` with three options:
  - "Choose from Library": `ImagePicker.launchImageLibraryAsync({ mediaTypes: 'images' as ImagePicker.MediaType, allowsEditing: true, aspect: APP_CONFIG.avatarAspect, quality: APP_CONFIG.avatarQuality, base64: true })`. On success, call `uploadAvatar(asset)`.
  - "Take Photo": first `ImagePicker.requestCameraPermissionsAsync()`, if granted `ImagePicker.launchCameraAsync(same options)`. On success, call `uploadAvatar(asset)`.
  - "Cancel" (style: 'cancel')
- `uploadAvatar(asset)`: copy exact pattern from ProfileSetup.tsx:
  - `const fileExt = asset.uri.split('.').pop()?.toLowerCase() ?? 'jpeg';`
  - `const filePath = \`${session.user.id}/avatar.${fileExt}\`;`
  - `await supabase.storage.from('avatars').upload(filePath, decode(asset.base64), { contentType: \`image/${fileExt}\`, upsert: true });`
  - Get public URL and setAvatarUrl
  - On error: `Alert.alert('Error', "Couldn't upload photo. Make sure the image is under 5MB and try again.")`
- Display name TextInput: `backgroundColor: COLORS.secondary, borderRadius: 12, height: 52, paddingHorizontal: 16, fontSize: 16, color: COLORS.textPrimary`, placeholder "Display name" in `COLORS.textSecondary`, `maxLength: APP_CONFIG.displayNameMaxLength`
- Below TextInput right-aligned: char count `<Text style={{ fontSize: 12, color: COLORS.textSecondary, textAlign: 'right', marginTop: 4 }}>{displayName.length}/{APP_CONFIG.displayNameMaxLength}</Text>`
- `isDirty = displayName.trim() !== originalDisplayName || avatarUrl !== originalAvatarUrl`
- `canSave = displayName.trim().length > 0 && isDirty && !saving`
- Save button: `<PrimaryButton title="Save Changes" onPress={handleSave} loading={saving} disabled={!canSave} />`
- `handleSave`: server confirmation pattern — `setSaving(true)`, `await supabase.from('profiles').update({ display_name: displayName.trim(), avatar_url: avatarUrl, updated_at: new Date().toISOString() }).eq('id', session.user.id)`. On error: `Alert.alert('Error', "Couldn't save your profile. Check your connection and try again.")`. On success: `router.back()`.
- ScrollView layout: `paddingHorizontal: 16, paddingTop: 32`
- Cancel: navigation back arrow in header (handled by Stack layout, no separate button)

  - Status row (centred, `marginTop: 8`): 10px circle View with `backgroundColor: COLORS.status[status]`, `borderRadius: 5`, `marginRight: 6` + status text at `fontSize: 16, color: COLORS.status[status]` + emoji tag inline if set
  - "Message [FirstName]" button: `<PrimaryButton title={\`Message ${profile.display_name.split(' ')[0]}\`} onPress={handleStartDM} />` with `marginTop: 24`
  - "Remove Friend" button: `<TouchableOpacity style={{ height: 52, justifyContent: 'center', alignItems: 'center', marginTop: 8 }}><Text style={{ fontSize: 16, color: COLORS.destructive }}>Remove Friend</Text></TouchableOpacity>`
- `handleStartDM`: call `supabase.rpc('get_or_create_dm_channel', { other_user_id: id })`, on success `router.push(\`/chat/room?dm_channel_id=${channelId}&friend_name=${encodeURIComponent(profile.display_name)}\` as never)`, on error `Alert.alert('Error', "Couldn't open chat. Try again.")`
