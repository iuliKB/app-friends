From src/theme/ThemeContext.tsx — useTheme() return shape:
```typescript
type ThemeContextValue = {
  colors: typeof DARK | typeof LIGHT;   // resolved palette — use colors.* everywhere
  isDark: boolean;
  theme: ThemePreference;               // 'light' | 'dark' | 'system'
  setTheme: (t: ThemePreference) => void;
};

// Usage (inside component body only):
const { colors } = useTheme();
```

Standard migration import swap:
```typescript
// BEFORE:
import { COLORS, SPACING, FONT_SIZE } from '@/theme';

// AFTER:
import { useTheme, SPACING, FONT_SIZE } from '@/theme';
// AND:
import React, { useMemo } from 'react';  // add useMemo
```

Sub-component pattern for MessageBubble (Pattern 3 note):
```typescript
// BEFORE — inner sub-component at file scope:
function QuotedBlock({ ... }) {
  return <View style={{ borderColor: COLORS.border }} />;  // can't useTheme() here at module scope
}

// AFTER — pass colors as prop:
function QuotedBlock({ ..., colors }: { ..., colors: ReturnType<typeof useTheme>['colors'] }) {
  return <View style={{ borderColor: colors.border }} />;
}

// In parent (MessageBubble export function):
const { colors } = useTheme();
// pass colors down: <QuotedBlock colors={colors} ... />
```

**Standard recipe:**
1. Change `import { COLORS, ... } from '@/theme'` → `import { useTheme, ... } from '@/theme'` (keep SPACING, FONT_SIZE, RADII, SHADOWS, etc.)

5. Replace all `COLORS.x` with `colors.x` — both in StyleSheet entries AND in inline JSX color props (`color={}`, `trackColor={}`, `thumbColor={}`, `tintColor={}`, `backgroundColor` set directly, `shadowColor={}`)

**SPECIAL CASE — MessageBubble.tsx (nested sub-components, Pattern 3 note):**
When you open `MessageBubble.tsx`, look for inner functions like `QuotedBlock`, `ReactionChip`, or any helper defined at the file scope (not exported). These helper functions cannot call `useTheme()` at module scope (not a valid hook location) and they are not React components with their own Provider context.

Resolution: add a `colors` prop to each inner helper function and pass the `colors` value from the parent `MessageBubble` component:

```typescript
// Helper signature:
type QuotedBlockProps = { ..., colors: ReturnType<typeof useTheme>['colors'] };
function QuotedBlock({ quote, colors }: QuotedBlockProps) {
  return <View style={{ borderLeftColor: colors.border }} />;
}

// In parent:
export function MessageBubble(...) {
  const { colors } = useTheme();
  // ...
  return <QuotedBlock quote={msg.quote} colors={colors} />;
}
```
