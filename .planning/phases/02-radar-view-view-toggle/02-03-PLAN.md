Scatter algorithm (from UI-SPEC — implement EXACTLY as specified):
```
Container: from onLayout → containerWidth, radarHeight (= 320 fixed)
Grid: 3 columns × 2 rows = 6 slots (max friends in radar)
Cell width  = containerWidth / 3
Cell height = radarHeight / 2   (= 160px)

For each friend slot (index 0-5):
  cellCol = index % 3
  cellRow = Math.floor(index / 3)
  cellOriginX = cellCol * cellWidth
  cellOriginY = cellRow * cellHeight

  bubbleRadius = BubbleSizeMap[friend.status with heartbeat fallback] / 2
  Bubble center X = cellOriginX + random(bubbleRadius + 8, cellWidth - bubbleRadius - 8)
  Bubble center Y = cellOriginY + random(bubbleRadius + 8, cellHeight - bubbleRadius - 8)
  (8px safety margin prevents clipping)

  bubbleLeft = bubbleCenter.X - bubbleRadius   (absolute left for positioning)
  bubbleTop  = bubbleCenter.Y - bubbleRadius   (absolute top for positioning)

Depth effect (applied after placement):
  upperHalf = bubbleCenterY < radarHeight / 2
  depthScale   = upperHalf ? 0.92 : 1.0
  depthOpacity = upperHalf ? 0.85 : 1.0
```

Positions are computed on mount AND whenever friends list changes. Re-randomized each mount (D-03).

4. `computeScatterPositions` pure function (outside component):
   ```typescript
   function computeScatterPositions(
     friends: FriendWithStatus[],
     containerWidth: number,
     radarHeight: number
   ): BubblePosition[] {
     const cellWidth = containerWidth / 3;
     const cellHeight = radarHeight / 2;
     return friends.map((friend, index) => {
       const heartbeat = computeHeartbeatState(friend.status_expires_at, friend.last_active_at);
       const status = heartbeat === 'dead' ? 'dead' : friend.status;
       const bubbleSize = BubbleSizeMap[status] ?? 36;
       const bubbleRadius = bubbleSize / 2;
       const cellCol = index % 3;
       const cellRow = Math.floor(index / 3);
       const cellOriginX = cellCol * cellWidth;
       const cellOriginY = cellRow * cellHeight;
       // Random placement within cell with safety margin
       const minX = cellOriginX + bubbleRadius + 8;
       const maxX = cellOriginX + cellWidth - bubbleRadius - 8;
       const minY = cellOriginY + bubbleRadius + 8;
       const maxY = cellOriginY + cellHeight - bubbleRadius - 8;
       // Clamp to ensure valid range (guard against very small cells)
       const centerX = minX < maxX ? minX + Math.random() * (maxX - minX) : cellOriginX + cellWidth / 2;
       const centerY = minY < maxY ? minY + Math.random() * (maxY - minY) : cellOriginY + cellHeight / 2;
       const upperHalf = centerY < radarHeight / 2;
       return {
         left: centerX - bubbleRadius,
         top: centerY - bubbleRadius,
         depthScale: upperHalf ? 0.92 : 1.0,
         depthOpacity: upperHalf ? 0.85 : 1.0,
       };
     });
   }
   ```
