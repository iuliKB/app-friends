**Bubble size map (exported for use in RadarView layout):**

export const BubbleSizeMap: Record<string, number> = {
  free: 64,
  maybe: 48,
  busy: 36,
  dead: 36,
};

**Implementation outline:**

1. Compute `heartbeatState = computeHeartbeatState(friend.status_expires_at, friend.last_active_at)` each render.

2. Determine `bubbleSize`:
   - heartbeatState === 'dead' → 36
   - Otherwise → BubbleSizeMap[friend.status] ?? 36

3. Determine outer opacity:
   - heartbeatState === 'dead' → 0.5
   - heartbeatState === 'fading' → 0.6
   - Otherwise → 1.0
   Apply `depthOpacity` multiplier on top: `finalOpacity = baseOpacity * (depthOpacity ?? 1.0)`

4. Determine scale: `finalScale = (depthScale ?? 1.0)` — applied via transform

5. Status resize animation (D-05): use `Animated.Value` initialized to `bubbleSize`. When `friend.status` or `heartbeatState` changes, run `Animated.timing({ toValue: bubbleSize, duration: 300, easing: Easing.inOut(Easing.ease), useNativeDriver: false, isInteraction: false })`. NOTE: `useNativeDriver: false` is required for width/height animations — document this explicitly with a comment.

6. PulseRing sub-component (ALIVE only):
   - Expand ring from `bubbleSize` to `bubbleSize * 1.7` using Animated.Value for size and opacity
   - Animated.loop with sequence: timing(expand 1200ms) + delay(600ms) then reset to start
   - `isInteraction: false` on all timing calls
   - `useNativeDriver: true` — use `transform: [{ scale }]` from 1.0 to 1.7 with opacity 0.7 → 0
   - Ring is absolutely positioned, centered behind the avatar

7. Status gradient (ALIVE Free/Maybe/Busy only, not FADING):
   - Use `LinearGradient` from 'expo-linear-gradient'
   - Position absolute, same dimensions as bubble, borderRadius = bubbleSize/2
   - Colors: `[statusColor at opacity, 'transparent']` where opacity is 0.30/0.25/0.20
   - direction: radial-ish using colors array `[rgba(...), 'transparent']`
   - Since expo-linear-gradient is linear (not radial), use: start={{ x: 0.5, y: 0.5 }}, end={{ x: 1, y: 1 }} for a center-to-corner feel

8. Interactions:
   - Pressable wrapping the whole bubble
   - onPress → tap handler: `supabase.rpc('get_or_create_dm_channel', { other_user_id: friend.friend_id })` → router.push to DM. Error: `Alert.alert('Error', "Couldn't open chat. Try again.")`
   - onLongPress (delayLongPress={400}) → `showActionSheet` with "View profile" + "Plan with [firstName]..."
   - hitSlop: when bubbleSize < 44 → `{ top: 4, bottom: 4, left: 4, right: 4 }` minimum to reach 44px effective area

9. Accessibility:
   - accessibilityRole="button"
   - accessibilityLabel: heartbeatState === 'fading' → `"${friend.display_name}, ${moodLabel}, fading. Tap to message, hold for more."`, else `"${friend.display_name}, ${moodLabel}. Tap to message, hold for more."`

10. Name label: `Text` below avatar, FONT_SIZE.sm (13px), FONT_WEIGHT.regular (400), COLORS.text.primary (FADING: COLORS.text.secondary), textAlign: center, maxWidth = bubbleSize + 16, marginTop: SPACING.xs (4px)

```typescript
import React from 'react';
import { Alert, Pressable, StyleSheet, View } from 'react-native';
import { useRouter } from 'expo-router';
import { COLORS, SPACING } from '@/theme';
import { AvatarCircle } from '@/components/common/AvatarCircle';
import { computeHeartbeatState } from '@/lib/heartbeat';
import { supabase } from '@/lib/supabase';
import type { FriendWithStatus } from '@/hooks/useFriends';

const STATUS_COLORS: Record<string, string> = {
  free: COLORS.status.free,
  maybe: COLORS.status.maybe,
  busy: COLORS.status.busy,
};

export function OverflowChip({ friend }: OverflowChipProps) {
  const router = useRouter();
  const heartbeatState = computeHeartbeatState(friend.status_expires_at, friend.last_active_at);

  const dotColor = heartbeatState === 'dead'
    ? COLORS.text.secondary
    : STATUS_COLORS[friend.status] ?? COLORS.text.secondary;
  const dotOpacity = heartbeatState === 'fading' ? 0.6 : 1.0;

  async function handlePress() {
    const { data, error } = await supabase.rpc('get_or_create_dm_channel', {
      other_user_id: friend.friend_id,
    });
    if (error || !data) {
      Alert.alert('Error', "Couldn't open chat. Try again.");
      return;
    }
    router.push(
      `/chat/room?dm_channel_id=${data}&friend_name=${encodeURIComponent(friend.display_name)}` as never
    );
  }

  return (
    <Pressable
      onPress={handlePress}
      hitSlop={{ top: 5, bottom: 5, left: 5, right: 5 }}
      style={styles.chip}
      accessibilityRole="button"
      accessibilityLabel={`${friend.display_name}. Tap to message.`}
    >
      <AvatarCircle size={34} imageUri={friend.avatar_url} displayName={friend.display_name} />
      <View style={[styles.dot, { backgroundColor: dotColor, opacity: dotOpacity }]} />
    </Pressable>
  );
}

const styles = StyleSheet.create({
  chip: {
    position: 'relative',
    marginRight: SPACING.sm,
  },
  dot: {
    position: 'absolute',
    bottom: 0,
    right: 0,
    width: 8,
    height: 8,
    borderRadius: 4,
  },
});
```
