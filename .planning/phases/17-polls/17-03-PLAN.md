```typescript
function handleOptionTap(optionId: string) {
  if (pollState?.myVotedOptionId === optionId) {
    // Tap currently selected option → un-vote (D-13)
    unVote();
  } else {
    // Tap unselected option → vote or change vote (D-11, D-12)
    vote(optionId);
  }
}
```

**OptionRow sub-component (or inline):**

Build a separate `OptionRow` function or inline inside the map. Each option row must:
1. Have its own `widthAnim = useRef(new Animated.Value(0)).current` — one per option
2. Run `useEffect` keyed on `[pollState?.myVotedOptionId, option.votes, pollState?.totalVotes]`:
   - When `pollState?.myVotedOptionId !== null`: animate width to `option.votes / (pollState?.totalVotes || 1)`, duration 300ms, `useNativeDriver: false`
   - When `pollState?.myVotedOptionId === null`: `widthAnim.setValue(0)` (instant reset — D-13)
3. Render:
   - `TouchableOpacity` wrapping the row; `minHeight: 44` (touch target); `activeOpacity: 0.7`
   - Radio circle: `View` 20px × 20px, `borderRadius: 10`, `borderWidth: 2`
     - Selected (myVotedOptionId === option.id): `backgroundColor: COLORS.interactive.accent, borderColor: COLORS.interactive.accent`
     - Unselected: `backgroundColor: 'transparent', borderColor: COLORS.border`
   - Label text: `FONT_SIZE.lg`, `FONT_WEIGHT.regular`, `COLORS.text.primary`, `flex: 1`, `marginLeft: SPACING.sm`
   - Progress bar (only visible when voted, i.e. `myVotedOptionId !== null`):
     ```jsx
     <View style={styles.barTrack}>
       <Animated.View style={[styles.barFill, {
         width: widthAnim.interpolate({ inputRange: [0, 1], outputRange: ['0%', '100%'] }),
         backgroundColor: isSelected ? COLORS.interactive.accent : COLORS.surface.overlay,
       }]} />
     </View>
     ```
   - Vote count (only visible when voted): `FONT_SIZE.sm`, `FONT_WEIGHT.regular`, `COLORS.text.secondary` — show `option.votes` as integer

**IMPORTANT:** If building OptionRow as a separate inner component, declare it outside the PollCard function body (not nested), or use `useCallback` — to avoid re-creating Animated.Value on each render. The safest approach is a named function component `OptionRow` declared before `PollCard` in the file, receiving props: `{ option: PollOption; isSelected: boolean; hasVoted: boolean; onPress: () => void }`.

**Card structure:**
```jsx
<View style={styles.card}>
  {/* Header: emoji + question */}
  <View style={styles.header}>
    <Text style={styles.emoji}>📊</Text>
    <Text style={styles.question}>{pollState.question}</Text>
  </View>
  {/* Divider */}
  <View style={styles.divider} />
  {/* Option rows */}
  {pollState.options.map((option) => (
    <OptionRow
      key={option.id}
      option={option}
      isSelected={pollState.myVotedOptionId === option.id}
      hasVoted={pollState.myVotedOptionId !== null}
      onPress={() => handleOptionTap(option.id)}
    />
  ))}
  {/* Footer divider + vote count */}
  <View style={styles.divider} />
  <View style={styles.footer}>
    <Text style={styles.footerText}>
      {pollState.totalVotes === 1 ? '1 vote' : `${pollState.totalVotes} votes`}
    </Text>
  </View>
</View>
```
