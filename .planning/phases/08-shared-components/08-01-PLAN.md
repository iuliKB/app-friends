**Props interface (export as `FABProps`):**
```typescript
interface FABProps {
  icon: React.ReactNode;       // Pass any icon component (e.g., <Ionicons name="add" size={24} color={COLORS.surface.base} />)
  label?: string;              // Optional label text — presence switches from circular to pill shape
  onPress: () => void;
  accessibilityLabel: string;
  size?: number;               // Optional custom size for icon-only variant (default: 56)
}
```

**Behavior:**
- Uses `useSafeAreaInsets()` for bottom positioning: `bottom: SPACING.xl + insets.bottom` (matches current `24 + insets.bottom`)
- Fixed `right: SPACING.xl` (matches current `right: 24`)
- `position: 'absolute'` always
- When `label` is provided: `flexDirection: 'row'`, `paddingHorizontal: SPACING.lg`, `paddingVertical: SPACING.lg`, `borderRadius: RADII.full` (pill shape) — icon and label side by side with `SPACING.sm` gap
- When no `label`: `width: size` (default 56), `height: size`, `borderRadius: RADII.full`, `alignItems: 'center'`, `justifyContent: 'center'`
- `backgroundColor: COLORS.interactive.accent`
- Spread `SHADOWS.fab` into the style
- Label text: `color: COLORS.surface.base`, `fontWeight: FONT_WEIGHT.semibold`, `fontSize: 15` (keep current value — no exact token match, use literal 15)

**Scale bounce animation on press:**
- Use `Animated.Value` for scale, starting at 1
- On `onPressIn`: `Animated.spring(scale, { toValue: 0.95, useNativeDriver: true, speed: 50, bounciness: 4 })`
- On `onPressOut`: `Animated.spring(scale, { toValue: 1, useNativeDriver: true, speed: 50, bounciness: 8 })`
- Wrap in `Animated.View` with `transform: [{ scale }]` containing a `TouchableOpacity` with `activeOpacity={0.8}`
- Or use `Pressable` with the animation — either approach works, but `TouchableOpacity` wrapping `Animated.View` is the simplest

**Implementation note:** Use `TouchableOpacity` inside the `Animated.View` (not `Animated.createAnimatedComponent`). The outer `Animated.View` handles the scale transform, inner `TouchableOpacity` handles press events.

Export both `FAB` and `FABProps` as named exports.
