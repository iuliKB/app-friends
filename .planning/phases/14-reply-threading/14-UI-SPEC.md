## Spacing Scale

Declared values (project tokens from `src/theme/spacing.ts`):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Avatar overlap offset, tight inline gaps |
| sm | 8px | Bubble padding, icon-label gap, separator height |
| md | 12px | Card section gaps, quoted block internal padding |
| lg | 16px | Standard screen padding, reply bar horizontal padding |
| xl | 24px | Large section spacing, overlay vertical offset |
| xxl | 32px | Screen header top, bottom sheet padding |

Exceptions:
- Context menu overlay: min 44px touch target height per action row (accessibility requirement)
- Quoted accent bar: 4px wide (RADII.xs width, not a spacing token — decorative dimension)
- Reply preview bar: 48px height (fixed, to keep it compact above SendBar)
- Dismiss button (×) in reply bar: 44px tap target (accessibility minimum)

## Color

All values from `src/theme/colors.ts`. Source: 14-CONTEXT.md code_context.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#1a1a1a` (COLORS.surface.base) | App background, chat screen background |
| Secondary (30%) | `#2a2a2a` (COLORS.surface.card) | Other-user message bubbles, SendBar background, reply bar background, context menu overlay background |
| Accent (10%) | `#f97316` (COLORS.interactive.accent) | Own-message bubble background, own-message quoted accent bar, reply bar "Replying to" label text, primary CTA icons |
| Destructive | `#ef4444` (COLORS.interactive.destructive) | Delete action label in context menu |
| Secondary text | `#9ca3af` (COLORS.text.secondary) | Preview text in quoted block (others' messages), timestamps, quoted block body text |
| Border | `#3f3f46` (COLORS.border) | Reply bar top border, quoted block separator |
| Overlay | `rgba(0,0,0,0.5)` (COLORS.overlay) | Context menu backdrop |

Accent reserved for:
- Own-message bubble background (`#f97316` fill)
- Own-message quoted accent bar (4px left border on quoted block inside own bubble)
- "Replying to [Name]" text label in reply preview bar
- Send and attachment button icons (existing pattern — no change)
- Ionicons color when enabled/active

Quoted accent bar color mapping (D-08 from CONTEXT.md):
- **Own message quoting anything**: accent bar color `COLORS.interactive.accent` (`#f97316`)
- **Others' message quoting anything**: accent bar color `COLORS.text.secondary` (`#9ca3af`)

Soft-deleted message placeholder color: `COLORS.text.secondary` (`#9ca3af`), italic style, body text "Message deleted."

## Component Inventory

New components and modifications required for this phase.

### New: BubbleContextMenu (inline overlay)

- Trigger: `onLongPress` on `TouchableOpacity` in `MessageBubble`
- Presentation: `Modal` with `transparent` background + `COLORS.overlay` backdrop (`rgba(0,0,0,0.5)`). Floating pill positioned absolutely above the tapped bubble.
- Shape: `RADII.lg` (12px) pill container, `COLORS.surface.card` background
- Actions row layout: horizontal flex, each action = icon + label, 44px minimum touch target
- Action rows: "↩ Reply" (all messages) | "📋 Copy" (all messages) | "🗑 Delete" (own messages only, `COLORS.interactive.destructive` text)
- Use Ionicons: `return-up-back` for Reply, `copy-outline` for Copy, `trash-outline` for Delete
- Dismiss: tap-outside backdrop (touchable `StyleSheet.absoluteFillObject` view)
- Z-index: rendered in Modal — above all content by default

### Modified: MessageBubble

- Add `onLongPress` prop alongside existing `onPress` (tap-for-timestamp is unchanged)
- Add `onReply`, `onCopy`, `onDelete` props for context menu actions
- Add quoted block rendering when `message.reply_to_message_id !== null`:
  - Quoted block sits above the bubble body text, inside the bubble container
  - Structure: left accent bar (4px wide, full height, `RADII.xs` border-radius) + content column (sender name line + preview text line)
  - Sender name: `FONT_SIZE.sm` (13px), `FONT_WEIGHT.semibold`, tinted color (see accent bar mapping above)
  - Preview text: `FONT_SIZE.sm` (13px), `FONT_WEIGHT.regular`, `COLORS.text.secondary`, max 1 line, ellipsis truncation
  - Quoted block background: `COLORS.surface.overlay` (`#ffffff14`) inside own bubble; `COLORS.surface.overlay` inside others' bubble
  - Quoted block padding: `SPACING.xs` (4px) vertical, `SPACING.sm` (8px) horizontal
  - Quoted block `RADII.xs` (4px) on all corners except left-side (where accent bar sits)
  - Tappable: `onPress` scrolls to original message
- Add deleted message placeholder:
  - When `message.body === null`: render "Message deleted." in `COLORS.text.secondary`, italic, `FONT_SIZE.md` (14px)
  - No context menu on deleted messages (skip `onLongPress` when body is null)
- Add highlight flash state:
  - `highlightAnim`: `Animated.Value(0)` → interpolates background color from transparent to `COLORS.interactive.accent` at 20% opacity → back to transparent
  - Duration: 1000ms, `useNativeDriver: false` (color interpolation)

### Modified: SendBar

- Add optional `replyContext` prop: `{ senderName: string; previewText: string; messageId: string } | null`
- Add optional `onClearReply` prop: `() => void`
- Reply preview bar renders between message list and existing send row when `replyContext` is non-null:
  - Height: 48px fixed
  - Background: `COLORS.surface.card`
  - Top border: 1px `COLORS.border`
  - Left padding: `SPACING.lg` (16px)
  - Content: "↩ Replying to [senderName]" — `FONT_SIZE.sm` (13px), `FONT_WEIGHT.semibold`, `COLORS.interactive.accent` | truncated preview — `FONT_SIZE.sm` (13px), `FONT_WEIGHT.regular`, `COLORS.text.secondary`, 1 line ellipsis
  - Dismiss × button: right side, `SPACING.lg` (16px) right padding, 44px touch target, `COLORS.text.secondary` icon

### New: Toast (scroll-to-original not-in-window)

- Render: `Animated.View` positioned at bottom of `ChatRoomScreen`, above `SendBar`, centered
- Background: `COLORS.surface.card` with `RADII.md` (8px) radius
- Text: "Scroll up to see the original message" — `FONT_SIZE.sm` (13px), `FONT_WEIGHT.regular`, `COLORS.text.primary`
- Padding: `SPACING.sm` (8px) vertical, `SPACING.lg` (16px) horizontal
- Animation: fade in over 200ms, hold 2000ms, fade out over 300ms — `Animated.timing` on opacity, `useNativeDriver: true`
- Non-blocking: does not prevent interaction with chat

## Interaction States

| Interaction | Trigger | Visual Result |
|-------------|---------|---------------|
| Long-press bubble | `onLongPress` (~500ms) | Context menu Modal appears above bubble; backdrop dims |
| Tap backdrop | `onPress` on backdrop View | Context menu Modal dismisses |
| Tap Reply | `onPress` on Reply action | Context menu closes; reply bar appears above SendBar |
| Tap Copy | `onPress` on Copy action | Context menu closes; text copied to clipboard (no visual feedback beyond system) |
| Tap Delete | `onPress` on Delete action | Context menu closes; message body set to null; bubble shows "Message deleted." placeholder |
| Tap × (reply bar) | `onPress` | Reply bar disappears; `replyContext` clears |
| Swipe down (reply bar) | `PanResponder` gesture | Reply bar dismisses; `replyContext` clears |
| Send with reply | `onSend` when `replyContext !== null` | Reply clears; message sent with `reply_to_message_id`; bubble renders with quoted block |
| Tap quoted block | `onPress` on quoted block | Scrolls FlatList to original message; flash highlight plays |
| Tap quoted block (not in window) | `onPress` on quoted block | Toast appears: "Scroll up to see the original message" |
| Flash highlight | After scroll-to-index | 1s background color pulse on target bubble |

## Accessibility

- All context menu action rows: minimum 44px height touch target
- × dismiss button in reply bar: minimum 44px touch target
- `accessibilityLabel` on each context menu action button: "Reply to message", "Copy message text", "Delete message"
- `accessibilityLabel` on quoted block: "Quoted message from [senderName]: [previewText]. Tap to scroll to original."
- `accessibilityLabel` on × button: "Cancel reply"
- Highlight flash: respects no `prefers-reduced-motion` API in React Native; animation is brief (1s) and non-critical
