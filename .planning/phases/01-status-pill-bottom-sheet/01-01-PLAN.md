Implementation details:

1. **Slide animation** — replicate FriendActionSheet exactly:
   - `translateY = useRef(new Animated.Value(300)).current`
   - On `visible=true`: `Animated.timing(translateY, { toValue: 0, duration: 250, useNativeDriver: true }).start()`
   - On `visible=false`: `translateY.setValue(300)` (instant reset, no slide-out — matches FriendActionSheet)

2. **Android BackHandler** — replicate FriendActionSheet exactly:
   ```typescript
   useEffect(() => {
     if (!visible) return;
     const handler = BackHandler.addEventListener('hardwareBackPress', () => {
       onClose();
       return true;
     });
     return () => handler.remove();
   }, [visible, onClose]);
   ```

3. **PanResponder swipe-down** — NOT in FriendActionSheet; add this:
   ```typescript
   const panResponder = useRef(
     PanResponder.create({
       onStartShouldSetPanResponder: () => false,
       onMoveShouldSetPanResponder: (_, gs) => gs.dy > 5,
       onPanResponderMove: (_, gs) => {
         if (gs.dy > 0) translateY.setValue(gs.dy);
       },
       onPanResponderRelease: (_, gs) => {
         if (gs.dy > 80 || gs.vy > 0.5) {
           Animated.timing(translateY, { toValue: 600, duration: 200, useNativeDriver: true }).start(onClose);
         } else {
           Animated.timing(translateY, { toValue: 0, duration: 150, useNativeDriver: true }).start();
         }
       },
     })
   ).current;
   ```
   Apply `{...panResponder.panHandlers}` to the drag-handle View only (not the full sheet — the MoodPicker's ScrollView must keep its own scroll).

4. **Auto-dismiss on commit** (D-02) — reactive pattern from MoodPicker lines 39-42:
   ```typescript
   const currentStatus = useStatusStore((s) => s.currentStatus);
   const prevStatusRef = useRef(currentStatus);
   useEffect(() => {
     if (prevStatusRef.current !== currentStatus && currentStatus !== null) {
       onClose();
     }
     prevStatusRef.current = currentStatus;
   }, [currentStatus, onClose]);
   ```

5. **Layout**:
   ```tsx
   <Modal visible={visible} transparent animationType="none" onRequestClose={onClose}>
     {/* Backdrop */}
     <TouchableWithoutFeedback onPress={onClose}>
       {/* eslint-disable-next-line campfire/no-hardcoded-styles */}
       <View style={StyleSheet.absoluteFillObject} /* backgroundColor: 'rgba(0,0,0,0.5)' */ />

     {/* Sheet */}
     <Animated.View style={[styles.sheet, { transform: [{ translateY }] }]}>
       {/* Drag handle — panResponder attached here */}
       <View {...panResponder.panHandlers} style={styles.dragHandleArea}>
         <View style={styles.dragHandle} />
       </View>
       <MoodPicker onCommit={onClose} />
     </Animated.View>
   </Modal>
   ```
