**4. Create `src/hooks/usePlans.ts`:**
- Export `usePlans()` hook returning `{ plans, loading, error, refreshing, fetchPlans, handleRefresh, createPlan }`
- `fetchPlans()`: Two-query approach (see RESEARCH.md Pattern 4):
  1. Query `plan_members` for `plan_id` where `user_id = session.user.id`
  2. Query `plans` with `.in('id', planIds).gte('scheduled_for', new Date().toISOString()).order('scheduled_for', { ascending: true })`
  3. Query `plan_members` for all those plan_ids with `.select('plan_id, user_id, rsvp, profiles ( id, display_name, avatar_url )')`
  4. Assemble `PlanWithMembers[]` and call `usePlansStore.getState().setPlans(result)`
- `createPlan(input: { title: string; scheduledFor: Date; location: string; invitedFriendIds: string[] })`:
  1. Insert into `plans` table: `{ title, scheduled_for: input.scheduledFor.toISOString(), location: input.location || null, created_by: session.user.id }` with `.select('id').single()`
  2. On success, insert into `plan_members`: creator row `{ plan_id: plan.id, user_id: session.user.id, rsvp: 'going' }` plus one row per invitedFriendId with `rsvp: 'invited'`
  3. Return `{ planId: string | null; error: Error | null }`
- Use `useFocusEffect` to refetch on tab focus (same pattern as `usePendingRequestsCount`)

**5. Create `src/hooks/usePlanDetail.ts`:**
- Export `usePlanDetail(planId: string)` returning `{ plan, loading, error, refetch, updateRsvp, updatePlanDetails, deletePlan }`
- `refetch()`: Query `plans` by id, then `plan_members` for that plan_id with profiles join. Assemble `PlanWithMembers`.
- `updateRsvp(newRsvp: 'going' | 'maybe' | 'out')`: Server confirmation pattern — `supabase.from('plan_members').update({ rsvp: newRsvp }).eq('plan_id', planId).eq('user_id', session.user.id)`. Return `{ error }`.
- `updatePlanDetails(updates: { title?: string; scheduled_for?: string; location?: string })`: `supabase.from('plans').update(updates).eq('id', planId)`. Return `{ error }`.
- `deletePlan()`: `supabase.from('plans').delete().eq('id', planId).eq('created_by', session.user.id)`. Return `{ error }`.
- Fetch on mount via `useEffect`.

**6. Create `src/hooks/useInvitationCount.ts`:**

  - Use `useFocusEffect` to refetch on focus
  - Return `{ count, refetch }`

function getNextRoundHour(): Date {
  const now = new Date();
  const next = new Date(now);
  next.setMinutes(0, 0, 0);
  next.setHours(now.getMinutes() > 0 ? now.getHours() + 1 : now.getHours());
  return next;
}
